## Quality Standards

- **Triangulation:** Every key claim needs 3+ independent sources
- **Confidence rating:** High / Medium / Low / Unverified on all findings
- **Source scoring:** Authority, recency, relevance, bias assessment
- **Iteration:** If gaps found at any stage, loop back to fill them
- **Attribution:** Every fact, statistic, and quote has a cited source
