# Callops Frontend Icons Directory

This directory should contain PWA icons in the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-180x180.png (Apple Touch Icon)
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Generate Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or create manually with your design tool.

## Design Guidelines

- Use the Callops gradient colors (primary: #7DD3C0, secondary: #4FD1C5)
- Include a phone icon or similar visual
- Make sure the 512x512 icon is maskable-safe (content within safe zone)
